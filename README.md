# LLM OWASP Scanner

An open-source security auditing tool that checks Large Language Model (LLM) 
applications against the OWASP Top 10 for LLM vulnerabilities.

## What It Does
Automatically scans and flags vulnerabilities across all 10 OWASP LLM risk 
categories:

- LLM01: Prompt Injection
- LLM02: Insecure Output Handling
- LLM03: Training Data Poisoning
- LLM04: Model Denial of Service
- LLM05: Supply Chain Vulnerabilities
- LLM06: Sensitive Information Disclosure
- LLM07: Insecure Plugin Design
- LLM08: Excessive Agency
- LLM09: Overreliance
- LLM10: Model Theft

## Who Is This For
- AI/ML Engineers securing LLM pipelines
- Security teams auditing AI applications
- Developers building on top of OpenAI, Anthropic, or open-source LLMs
- Red teamers testing LLM robustness
